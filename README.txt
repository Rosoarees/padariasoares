Este projeto é uma demonstração de uma aplicação web para a "Padaria Soares", construída com foco em HTML semântico, CSS moderno (com design responsivo) e JavaScript puro para interatividade. O objetivo é simular um site completo de padaria, agora com uma arquitetura multi-página para melhor organização e escalabilidade.

Funcionalidades Principais

Navegação Multi-página: O site foi refatorado para incluir páginas dedicadas para o Cardápio, Contato e Pedidos Especiais, melhorando a organização e a experiência do usuário.
Design Responsivo: Layout adaptável a diferentes tamanhos de tela (desktops, tablets e celulares).
Catálogo de Produtos: Exibição de pães, bolos e salgados com detalhes e preços, agora em uma página de Cardápio dedicada.
Sistema de Carrinho de Compras: Adicione produtos, ajuste quantidades, remova itens e visualize o total do pedido.
Calculadora de Pedidos: Uma ferramenta interativa para calcular o valor de pedidos com base em preço unitário, quantidade, impostos e taxas de entrega, incluindo conversão de moedas.
Favoritos: Funcionalidade para marcar e gerenciar produtos favoritos.
Sistema de Busca: Pesquisa dinâmica de produtos para encontrar itens rapidamente.
Modais Interativos: Modais para detalhes de produtos e cadastro de clientes, integrados com validação de formulários.
Formulário de Feedback: Um formulário para coleta de opiniões dos clientes, agora na página de Contato.
Painel Administrativo (Simulado): Uma seção oculta com estatísticas e controles de preços, para fins demonstrativos.
Otimização Básica de SEO: Uso de tags semânticas, `alt text` para imagens, atributos ARIA para acessibilidade e meta tags relevantes.
Animações e Transições: Elementos visuais com transições suaves para uma experiência mais agradável.

Tecnologias Utilizadas

HTML5: Estrutura semântica e acessível.
CSS3: Estilização moderna, responsividade (Flexbox e Grid CSS) e variáveis CSS.
JavaScript (ES6+): Lógica de negócios e interatividade.


Este projeto foi desenvolvido com uma forte ênfase em acessibilidade (ARIA attributes, keyboard navigation) e semântica HTML, garantindo que o conteúdo seja compreensível e navegável para o maior número de usuários possível.
